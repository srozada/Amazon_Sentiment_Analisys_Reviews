# Amazon Sentiment Reviews

## Project Description

A **Data Science** project focused on sentiment analysis of product reviews on the **Amazon** platform. The system classifies user ratings to extract actionable insights for both the recommendation system and sellers.

## Objective

Develop a Machine Learning model capable of:

- **Boosting the visibility** of products with higher ratings within Amazon's recommendation system.
- **Penalizing the visibility** of products with negative ratings, improving the end-user experience.
- **Generating automated feedback** for product creators, helping them identify areas for improvement based on customer reviews.

## Scope

The project covers everything from data collection and preprocessing of reviews to the training and evaluation of sentiment classification models, with the goal of integrating into an intelligent recommendation pipeline.
